Recetas que se adaptan a tu día

API de Recetas

Bienvenidos a la API de ¿Y HOY QUÉ?,
una aplicación web que te ayuda a encontrar recetas
personalizadas según los ingredientes que tienes en casa.
Podrás buscar, filtrar, guardar recetas favoritas, dejar comentarios y mucho más.

Rutas de la API

Loguear usuario

  • Loguin siendo usuario: POST http://localhost:3000/api/login 
                          
                        {
                          "email": "juan@gmail.com",
                          "password": "12345678"
                        }

    • Ruta recetas

  • Obtener todas las Recetas: GET http://localhost:3000/api/recipes
  • Obtener receta por ID: GET http://localhost:3000/api/recipes/:id (66f9f8e903c4f3cff9d02ae8)
  • Obtener receta por Categoría: GET http://localhost:3000/api/recipes/category/Main Course
  • Obtener receta por Dificultad: GET http://localhost:3000/api/recipes/difficulty/Easy
  • Buscar recetas por Nombre: GET http://localhost:3000/api/recipes?nombre=Asado
  • Buscar recetas por Página y Límite: GET http://localhost:3000/api/recipes?page=1&limit=5
  • Buscar recetas por Ingredientes: GET http://localhost:3000/api/recipes?ingredients=carne,cebolla

    • Ruta comentarios

  • Agregar comentario a una receta: POST http://localhost:3000/api/comments 
                          {
                        "recipeId": "66f9f87c03c4f3cff9cff145", 
                        "comment": "Esta receta es excelente!",
                        "rating": 5
                      }
  • Ver comentarios de una receta: GET http://localhost:3000/api/comments/:recipeId(66f9f86203c4f3cff9cfe322)

    • Acceso rol "admin"

  • Loguin siendo admin: POST http://localhost:3000/api/login
